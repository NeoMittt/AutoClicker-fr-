# 🖱️ AutoClicker

AutoClicker is a simple and user-friendly tool that automates left and right mouse clicks, scroll wheel actions, and keyboard key presses.

## 🎯 Features

- Auto-clicking for left, right and middle mouse buttons.
- Support for automated keyboard key presses.
- Flexible settings for click speed and frequency.
- Intuitive and easy-to-use interface.

## 💻 Requirements

- Windows 7 / 8 / 10 / 11
- .NET Framework 4.8 or later

## 🛠️ Usage

1. Choose the type of click (left button, right button, scroll).
2. Set the frequency and number of clicks.
3. Press the "Start" button to begin auto-clicking.
